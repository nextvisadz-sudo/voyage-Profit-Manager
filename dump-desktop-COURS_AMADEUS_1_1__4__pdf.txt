 Amadeus :
AMADEUS Air offre un accès permanent à plus 480 compagnies aériennes dans le
monde.
Les différents acteurs qui interviennent lors d’une réservation :
 La compagnie aérienne.
 Amadeus.
 L’agent de voyage.
 Le client.
 Signature du poste du travail :
1. J’ouvre la page Amadeus.
2. Je clique sur « Lancer » « Amadeus selling plat form »
3. Je tape le mot de passe (Personnel de l’agent)
4. « OK » pour valider.
5. J’active la page « commande »
6. En cas d’une déconnexion je tape le format :
a. En mode production ou (Product mode) :
J I signature du poste/S U- mot de passe
Exp: J I 2010 AM/S U - mohamed ENTRER
b. En mode training :
J J signature du poste /SU- mot de passe
Exp: J J 2010 AM/S U- mohamed

Leçon 4: LES TYPES DES VOLS
Application :
>SN3 JANTUN DXB
- Il y a 2 vols un direct et l’autre avec escale.
Les types :
a) Vol Non stop : Avec le même numéro de vol et le même type d’équipement.
b) Vol Direct : vol avec escale avec le même N° du vol et le même type d’équipement.
c) Vol Par correspondance : vol avec escale et un numéro du vol et un équipement différent.
Exercice :
>SN 5 JAN TUN DJE 4 vols directs

-Application:

>SN 6 Mar PAR MIL

2 lettres
- AP * AZ Partage de code location d’avion avec équipage, une compagnie loue un appareil et un
équipage auprès d’une autre compagnie aérienne et commercialise le vol sous son propre
Code à 2 lettres.

 Pour dupliquer un « PNR » et reproduire les mêmes segments de vols on tape le

B. SPLIT OU SEPARATION DE DOSSIER
Introduction :
Le split, ou séparation de dossier permet de diviser manuellement un dossier passager
initialement homogène.
Cette transaction est à effectuer si les passagers du dossier n’empruntent plus le même
itinéraire,
Les mêmes vols…
Fonctionnement :
La fonction split va générer alors deux dossiers :
Le dossier d’origine (appelé parent)
Le dossier Associé
Application : Départ 20Feb TUN DJE Retour le 28 FEB
>SN 20 FEB TUN DJE
>SS2Y1
>SN 28 FEB DJE TUN
>SS2Y1
>NM2 AOUINI/ MAKREM/MOHAMED
>AP 22501817/P1
>AP 22501817/P2
>TKOK
>RF ML
>ER

Leçon 13 : REBRIQUES
COMPLEMENTAIRES
 Introduction :
Les rubriques complémentaires servent à signaler certaines informations concernant le
passager
 Remarque générale.
 Remarque confidentielle.
 Message d’information.
 Demande de services spéciaux.
1. Remarque générale :
• Permet de noter des informations supplémentaires concernant le passager ou ses
réservations.
• Peut être lu par tout site autorisé à visualiser le dossier
• Ne génère pas de message à la compagnie aérienne.
EXP : >RM ok pour l’émission de billet ou bon de commande ok.
2. Remarque confidentielle :
La remarque confidentielle ne peut être lue que par le site créateur de la remarque (l’agence
de voyages)
EXP :>RC STE LG
Les sites non autorisés à lire la remarque verront : RC RESTRICTED
3. Message d’information :
Permet de transmettre à la compagnie aérienne une information concernant le passager.
EXP1 : Pour indiquer que le passager est un client important tapez le format :
> OS AA VIP PRESEDENT OF IBM
Code Nom type de client société
Compagnie
EXP2: Pour communiquer le N° de billet émis à une compagnie, ne travaillant pas elle-même
sur Amadeus :
>OS AA TKNO 001 1234567890
YY: ANY IATA (toutes les compagnies aériennes)
4. Demande de service spécial :
La rubrique SR (SPECIAL SERVICE) permet de transmettre à une compagnie une demande de
service spéciale pour un ou plusieurs passagers.

Leçon 16 : VENTE D’UN PASSAGER « UM »
Un enfant dont l’âge est entre 4 ou 5ans selon les compagnies aériennes et 12ans,
voyageant seul est appelé un UM (Unaccompagnied Minor).
Les règles pour la vente d’un UM sont les suivantes :
• Code de transaction spécifique.
• Sur un vol confirmé uniquement (pas de liste d’attente)
• Une vente UM ne peut être faite dans un dossier comportant d’autres types de
passager.
• Les contacts des personnes accompagnant et réceptionnant l’enfant doivent être
mentionnés.

>SS1Y1
>SN3 JAN TLS TUN
>SS1Y1
